# Data Policy

OpenMonitor Science Kits does not host large raw scientific archives.

The repository contains:

- notebooks;
- readers;
- schemas;
- documentation;
- tiny sample files.

The OpenMonitor web platform may generate derived exports such as CSV tables,
FITS tables, plots, movies, and metadata JSON files. The kits here explain how
to read those exports.

Underlying observations remain with their official providers.
