# Citing OpenMonitor Science Kits

If a notebook or reader from this repository helped your work, cite
OpenMonitor and the underlying scientific data providers.

Suggested sentence:

> This analysis used OpenMonitor Science Kits, the public reproducibility
> companion to OpenMonitor.org.

For AIA ROI light curves, also cite the underlying AIA/SDO and JSOC data source
according to their official guidance.

This repository does not replace provider citations.
