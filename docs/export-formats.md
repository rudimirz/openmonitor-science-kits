# Export Formats

OpenMonitor exports may include several file types. Not every user needs every
file.

Common files:

- CSV: easiest for spreadsheets and quick Python use.
- FITS table: best for scientific reuse when metadata matters.
- PNG: static visual preview.
- GIF: movie or animated context.
- JSON metadata: reproducibility, support, and programmatic inspection.

Kit-level README files explain the specific files for each workflow.
